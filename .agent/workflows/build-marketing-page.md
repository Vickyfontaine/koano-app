# Workflow: build a KOANO marketing page

## Before writing any code
1. Read CLAUDE.md completely
2. Activate koano-design skill
3. Identify which section of the homepage wireframe you are building
4. Confirm section order matches Section 06 of CLAUDE.md exactly

## Building order for every page
1. HTML structure and section order first — no styling yet
2. Apply Tailwind classes and CSS variables
3. Add approved copy verbatim from Section 10 of CLAUDE.md
4. Add Framer Motion entrance animations last
5. Render [COPY TBD] placeholders as italic faint text in dashed containers

## Never do
- Invent copy not in CLAUDE.md
- Add sections not in the wireframe
- Use dark backgrounds
- Use placeholder grey boxes
- Use any font other than Neue Montreal
